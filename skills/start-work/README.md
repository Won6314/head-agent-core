## Why It Exists

Long-running work needs a durable agreement that survives interruption and compaction without forcing every outcome into one document template. This skill establishes that work model and its recovery contract.

## Boundary

The skill defines the role of the run and its checklists. Each project chooses its session location, entry point, evidence sources, and domain-specific planning workflow.
