## Why It Exists

Delegation succeeds when one owner receives one coherent result with enough authoritative context to act independently. This skill separates that reasoning model from any particular worker runtime or project roster.

## Boundary

The skill shapes ownership, context, dependencies, and integration. The [`agent-task` MCP](../../mcp/agent-task/README.md) executes worker operations, while the project supplies specialist availability and routing.
