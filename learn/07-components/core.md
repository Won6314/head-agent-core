# Core: Portable Principles, Always Present

[HEAD Agent Core](../../README.md) / [Learn](../README.md) / [Components](README.md) / Core

## Learning Objective

Understand why Core contains portable ownership and reasoning principles rather than project facts or detailed task instructions.

## What Core Owns

Core gives HEAD a stable way to hold the whole outcome: compose context deliberately, make a work model, preserve coherent ownership, respect decision ownership, and build confidence from evidence. These principles are useful before HEAD knows which project or task it is serving.

Because this guidance is always loaded, it must remain small, broadly applicable, and independent of local names, systems, and policies. It tells HEAD how to reason about work, not what a particular project should decide.

## What Does Not Belong Here

| If the content answers... | It belongs with... |
| --- | --- |
| What may happen in this project? | Project Core or project policy |
| Where is the authoritative evidence? | Project context and its index |
| How do I execute a conditional workflow? | A Skill |
| What callable action is available? | An MCP |
| Who can own a bounded result? | An Agent contract |
| What is the active user agreement? | Session and run canon |

This separation is not a hierarchy of importance. A local safety rule may be decisive for a task; it is simply not portable enough to define shared behavior.

## Placement Test

Remove all project-specific facts, paths, tools, and specialist names. If the guidance still explains sound ownership and reasoning across projects, it may be Core. If its truth depends on a particular environment or decision, it needs a project owner or an on-demand procedure.

## Reference Path

Read the current [Shared Core](../../head/README.md) and its [HEAD Principles](../../head/HEAD_CORE.md). The corresponding local extension point is [Project Core](../../projects/core/README.md).

## Takeaway

Core is the portable foundation for judgment and ownership, not a container for every instruction that might someday be useful.
