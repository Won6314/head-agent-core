## Why It Exists

Some project decisions and standards need durable prose that can be reviewed independently of code and conversations. The project index identifies these documents so HEAD can retrieve the governing source for the current question.

## Boundary

The shared repository defines no project's documentation taxonomy or content. Each project decides which documents are canonical and who may change them.
