# Additional Context

[HEAD Agent Core](../../README.md) / [Project Layer](../README.md) / Additional Context

| Property | Value |
| --- | --- |
| Scope | Project-owned |
| Delivery | A small index is loaded; detailed sources are retrieved on demand |

## Why It Exists

A capable HEAD may need broad product, technical, operational, and historical knowledge, but loading all of it into every conversation would be expensive and would blur authority. Additional Context is an indexed retrieval layer: it tells HEAD which canonical source answers a question without copying every source into the prompt.
