## Why It Exists

Independent review needs a stable evidence standard and a strict separation between judgment and repair. A shared contract lets validation remain comparable even when the reviewed artifact and project domain change.

## Boundary

The shared role defines review authority and evidence discipline. The project supplies the governing requirements, primary sources, review modes, and any domain-specific restrictions.
